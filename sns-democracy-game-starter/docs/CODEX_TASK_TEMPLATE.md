【完成条件】

## 依頼の書き方の例

### 良い例

```text
【今回の作業名】
第1章・SNSタイムライン画面の表示実装

【今回してよいこと】
- story.json内の仮投稿データを読み込み、カード形式で表示する。
- iPad横向きで読みやすい文字サイズ・余白に調整する。
- 画面遷移の仮ボタンを追加する。

【完成条件】
- 投稿カードが3件以上表示される。
- 長文投稿が画面からはみ出さない。
- 戻る・次へが操作できる。
```

### 避ける例

```text
第1章をもっと面白くして、SNSっぽくして。
```

理由：作業範囲・完成条件・変更禁止事項が曖昧であり、Codexが教育的内容まで勝手に補う余地が大きい。
