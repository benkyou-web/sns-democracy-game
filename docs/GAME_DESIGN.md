# SNSノベルゲーム：ゲーム設計書 v0.1

## 1. 仮題

**『タイムラインの外側』**

副題：見えている世界は、本当に全部だろうか。

※正式タイトルは試作の方向性が定まってから決定する。

## 2. 教材の位置づけ

- 対象：高等学校「公共」
- 主題：情報社会、民主主義、意見形成、対話、人間関係
- 形式：HTML/CSS/JavaScriptで動作するWebノベルゲーム
- 配布方法：GitHub Pages等で公開し、授業ではURLを配布する
- 想定端末：生徒用iPadのブラウザ

## 3. このゲームで生徒に考えさせたいこと

このゲームは、SNSの利用を禁止したり、インフルエンサーや動画投稿者を一律に否定したりするためのものではない。

生徒が、次のことに気づくことを目指す。

- 自分が「当たり前」「正しい」「みんなそう思っている」と感じていることは、繰り返し見た情報や周囲の反応によって形づくられているかもしれない。
- 再生数、共感数、コメント欄の雰囲気、話し方の勢い、相手を言い負かしたように見える演出は、意見の正しさそのものと同じではない。
- 自分と異なる意見を持つ人を、すぐに敵・非常識・悪意ある人と決めつけないことが、民主社会で他者と共に生きるために重要である。
- 自分の意見を持つことと、自分の意見だけに閉じこもらないことは両立できる。

## 4. 中心問い

### 全体の中心問い

**意見が異なる人と共に生きる社会で、私たちはどのように情報を受け取り、考え、関わればよいのだろうか。**

### プレイ中の問い

**自分が「当たり前」だと思っている考えは、どのような情報や人間関係によって形づくられているのだろうか。**

### プレイ後の問い

**自分と異なる意見を持つ人と、関係を切らずに社会の問題を考えるために、何が必要だろうか。**

## 5. 概念の扱い

### フィルターバブル

利用者の興味・行動・反応などに応じて、似た情報が優先的に届きやすくなり、異なる情報や立場に触れにくくなる現象。

### エコーチェンバー

似た意見を持つ人々の間で、同じ考えが繰り返し共有・肯定されることで、その考えが一層正しいもの・多数派の常識であるように感じられる状態。

### 教材上の扱い

プレイ中に用語を説明して理解させるのではなく、選択に応じて見える投稿・コメント欄・人間関係を変化させることで、まず体験させる。

用語は、プレイ後に他者の結果と比較した後で導入する。

## 6. 試作版の範囲

### 舞台

高校の学校祭に関するルール変更をめぐり、ある動画投稿がSNSで拡散される。

動画は「学校が生徒の自由を奪っている」と強く批判しており、コメント欄には「正論」「論破」「先生たちは反論できない」といった反応が多く並ぶ。

一方で、ルール変更には、準備負担、安全面、過去のトラブル、実行委員の苦労など、動画では十分に扱われていない事情もある。

### プレイ時間

15〜20分程度。

### 主要人物

- 主人公：高校生。学校祭に関心はあるが、詳しい事情をすべては知らない。
- 親友：動画を共有する。学校のルールに息苦しさを感じており、動画の主張に共感している。
- 実行委員：ルール変更に関わった生徒。安全や運営負担を考えているが、説明不足への迷いもある。
- 担任または顧問：学校側の事情を知る大人。生徒の意見を抑えたいわけではないが、対立が深まることを心配している。
- 動画投稿者：学校外の発信者。分かりやすく強い言葉で問題を語り、支持を集める。善悪の単純化を避け、問題提起の鋭さと見落としの両方を持たせる。

### 試作版で扱う出来事

1. 動画が共有される。
2. グループチャットで意見が盛り上がる。
3. 主人公が最初の対応を選ぶ。
4. 選択により、見える投稿・人からの連絡・得られる情報が変わる。
5. 小さな対立またはすれ違いが起きる。
6. 最後に、主人公の選択履歴に応じた振り返りが表示される。

## 7. ゲーム構造の原則

### 共通の本筋と、個別化された世界

大きな事件の順番は全プレイヤーで共通とする。

一方、次の要素は選択履歴に応じて変化する。

- タイムラインに現れる投稿
- コメント欄の雰囲気
- 相談や連絡をしてくる人物
- 主人公が聞ける事情
- 友人との距離感
- 最後の振り返りの問い

### 目指す体験

生徒が、他者とプレイ結果を比べたときに、次のように気づける構造を目指す。

> 同じ事件を扱っていたのに、自分が見た投稿や聞いた話は他の人と違った。
> 現実のSNSでも、私たちはそれぞれ少しずつ違う世界を見ているのかもしれない。

## 8. 裏で管理する要素

数値は生徒に直接表示しない。画面に見える情報と人物の反応で間接的に表現する。

| 要素 | 意味 | 表現例 |
|---|---|---|
| 情報の幅 | 異なる立場や背景事情に触れられている度合い | 一次資料、当事者の話、別立場の投稿が見える |
| 同調への傾き | 似た意見や強い言葉に囲まれやすい度合い | 「正論」「論破」などの投稿・動画がおすすめに増える |
| 近い関係の信頼 | 親しい人物が本音を話してくれる状態 | DMで迷いや不安を相談される |
| 発信の影響 | 主人公の投稿や発言が広がりやすい状態 | 返信、拡散、反論、期待が増える |
| 対話の余地 | 異なる意見の人と関係を保てる状態 | 話を聞く機会、直接会話する場面が残る |

## 9. 試作版の完成条件

- 15〜20分で一周できる。
- 少なくとも3回の選択場面がある。
- 選択により、SNS投稿または人物の反応が変化する。
- 少なくとも2種類の結末または振り返り画面がある。
- 選択肢に明白な正解・不正解が見えない。
- プレイ中は「フィルターバブル」「エコーチェンバー」を用語として説明しない。
- プレイ後に、他者との比較・概念学習・振り返りへ接続できる。
- iPadのブラウザで操作できる。
- 個人情報を取得しない。

## 10. 今後の拡張方針

試作版で、選択肢の自然さ、画面の使いやすさ、振り返りの深さを検証する。

本番版では、以下を段階的に追加する。

- 第2章以降の人間関係の複雑化
- 異なる立場の人物との直接対話
- 動画の切り取りや誤解が明らかになる場面
- 主人公自身の発信が拡散される場面
- 複数のエンドと、選択履歴に応じた個別振り返り
- 授業用の比較・対話・記述活動
