# SNSノベルゲーム：選択肢設計原則 v0.1

## 1. この文書の目的

このゲームの選択肢は、生徒に「正しい答え」を当てさせるためのものではない。

選択を通して、生徒が自分の価値観、情報の受け取り方、人間関係への向き合い方を振り返るためのものである。

そのため、選択肢は教育的に望ましい行動と望ましくない行動を露骨に並べない。

## 2. すべての選択肢が満たす条件

各選択肢には、必ず次の3点を持たせる。

1. **選ぶ理由があること**
   - その選択をする人物の気持ちや立場が理解できる。

2. **その場で守れるものがあること**
   - 友情、慎重さ、場の安定、自分の意見、事実確認、当事者への配慮など、何かしらの価値を守れる。

3. **見落とす可能性があること**
   - 別の立場、誤解の広がり、沈黙の意味、関係の変化、発信の影響、必要な事実などを見落とす余地がある。

## 3. 避けるべき選択肢

以下のように、教材としての正解・不正解が分かりやすい選択肢は使わない。

- 「感情的になってすぐ拡散する」
- 「冷静に一次情報を確認する」
- 「相手の話を聞かずに否定する」
- 「みんなに流される」
- 「正しい情報だけを信じる」
- 「根拠のないうわさを広める」

これらは、プレイヤーに考えさせる前に、作り手の意図を見せてしまう。

## 4. 望ましい選択肢の型

### 型A：公の場で言う／個別に聞く／保留して調べる

状況：親友が、学校のルールを批判する動画をグループチャットに共有した。

- グループチャットで、別の事情もあるのではないかと書く。
- まず親友に個別で連絡し、何が気になったのか聞く。
- 自分でも資料や経緯を確認してから、発言するか決める。

この三択では、

- 公の場で言うことは、誤解の広がりを止める可能性があるが、友人を追い詰めるかもしれない。
- 個別に聞くことは、関係を守りやすいが、グループ内の議論には影響しにくい。
- 保留して調べることは、軽率な発言を避けられるが、沈黙がどのように受け取られるかは分からない。

という形で、どの選択にも理由と限界がある。

### 型B：発信する／直接話す／見守る

状況：ある投稿が急速に拡散され、クラス内でも対立が深まりつつある。

- 自分の考えを短く投稿し、議論に加わる。
- 関係する人に直接会って、話を聞く。
- 今は議論に加わらず、どのように状況が動くかを見る。

### 型C：支持する側に近づく／反対側に近づく／立場を固定しない

状況：意見が二つに割れている。

- 自分に近いと感じる意見を出している人たちと話す。
- 反対の意見を持つ人が何を心配しているのか聞く。
- どちらにも加わらず、必要な情報を集める。

この型でも、どれかを「中立だから正しい」と扱わない。立場を固定しないことにも、責任を引き受けない・考えを伝えられないという限界を持たせる。

## 5. 選択の直後に結論を出さない

選択の直後に、次のような表示はしない。

- 「信頼が上がった」
- 「正しい判断だった」
- 「情報リテラシーが上がった」
- 「悪い選択をした」

代わりに、以下のような間接的な変化で示す。

- タイムラインに流れる投稿が変わる。
- コメント欄の雰囲気が変わる。
- 既読の付き方や返信の温度が変わる。
- 特定の人物からDMが届く、または届かなくなる。
- 後の場面で、以前の選択の影響が見えてくる。

## 6. 人物を単純化しない

### 動画投稿者

- 問題提起の鋭さや分かりやすさを持つ。
- しかし、事情の省略、強い断言、切り取りによる影響も持つ。
- 単なる悪人・詐欺師・無知な人として描かない。

### 親友

- 動画に共感する理由がある。
- 自由を尊重したい、不公平を感じている、説明されないことへの不信感があるなど、納得できる背景を持つ。
- 安易に「流されやすい人」と描かない。

### 実行委員

- ルールの必要性を理解している。
- しかし、説明不足や生徒への伝え方に悩んでいる。
- 「正しい学校側」の代表にしない。

### 大人の関係者

- 対立を避けたい、責任を持ちたいという事情がある。
- 同時に、生徒の意見を十分に聞けていない可能性も持つ。

## 7. フィルターバブルとエコーチェンバーの表現方法

### フィルターバブル

- 特定の動画を最後まで見る、共有する、共感するなどの行動後に、似た論調の投稿が増える。
- 反対意見が「消える」のではなく、見えにくくなる。
- 後になって、別のプレイヤーが見ていた投稿を知ることで、自分の情報環境の偏りに気づける。

### エコーチェンバー

- グループチャットやコメント欄で、同じ意見への賛同が繰り返される。
- 「みんなが言っている」「普通はそう思う」といった表現が増える。
- ただし、参加者が悪意で同調しているようには描かない。安心感、共感、怒り、孤立への不安など、人間的な理由を持たせる。

## 8. 最後の振り返りの原則

結末では、生徒を診断・採点しない。

「あなたは偏っている」「あなたは正しい」と断定せず、選択履歴に応じた問いを表示する。

### 強い発信・同調を選びやすかった場合

> あなたの画面には、「正論」「論破」と呼ばれる投稿が多く流れていました。  
> その意見を正しいと感じた理由は、根拠でしたか。  
> それとも、繰り返し見たことや、周囲の反応でしたか。

### 慎重な保留・沈黙を選びやすかった場合

> あなたは、軽率に断定しないことを選んできました。  
> その慎重さによって守れたものは何でしょうか。  
> 反対に、誰かの誤解が広がる場面で、言えなかったことはありませんか。

### 近い関係を優先しやすかった場合

> あなたは、近い人との関係を大切にしてきました。  
> その選択によって守れたものは何ですか。  
> 逆に、聞きにくくなった意見や、見えにくくなった事情はありませんでしたか。

### 異なる立場を聞こうとした場合

> あなたは、自分と異なる立場の人の話を聞いてきました。  
> それによって何が見えるようになりましたか。  
> 理解することと、賛成することは同じでしょうか。

## 9. 最終確認チェックリスト

新しい選択肢を追加・修正する前に、必ず確認する。

- [ ] どの選択肢にも選ぶ理由があるか。
- [ ] どの選択肢にも、その場で守れる価値があるか。
- [ ] どの選択肢にも、見落とす可能性があるか。
- [ ] 一つだけが明らかに模範解答に見えていないか。
- [ ] 特定の人物・立場を単純な悪役にしていないか。
- [ ] その場で結論を採点していないか。
- [ ] 後の場面で、選択の意味を振り返れるか。
- [ ] 民主主義・対話・人間関係への問いにつながっているか。
